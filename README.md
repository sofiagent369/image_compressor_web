# Image Compressor Web 🚀

Aplicación web de compresor de imágenes online con interfaz moderna tipo dashboard. Incluye areas de input/output, historial de uso guardado en `localStorage`, exportar resultados como archivo y tema claro/oscuro.

## Características Principales
- **Interfaz Moderna**: Diseño responsive y accesible.
- **Compresión de Imágenes**: Utiliza la API Canvas para comprimir imágenes.
- **Historial de Uso**: Guarda el historial en `localStorage`.
- **Tema Claro/Oscuro**: Cambia fácilmente entre temas.
- **Exportación de Resultados**: Exporta la imagen comprimida como archivo.
